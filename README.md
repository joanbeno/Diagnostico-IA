## ¿Qué hace?

- Evalúa el nivel de madurez en IA de cada persona del equipo
- Detecta fricciones reales en el uso de ChatGPT
- Genera un perfil personalizado con recomendaciones accionables
- Entrega un código personal para recuperar resultados y acceder a contenido exclusivo
- Incluye guía interactiva con plantillas, plan de acción y módulos de aprendizaje
